# Projet FleetControl - Gestion de flotte et maintenance automobile

## Contexte

FleetControl est une entreprise spécialisée dans la gestion de flottes de véhicules pour des sociétés de transport, livraison et maintenance.  
Le projet consiste à concevoir et implémenter une base de données centralisée pour gérer :

- Les véhicules et leur catégorisation
- Les conducteurs et leurs affectations
- Les trajets et missions
- Les incidents et interventions
- Les pièces utilisées, fournisseurs, techniciens
- Les assurances, contrôles techniques et contrats associés

## Phase 1 : Modélisation

- MCD (Modèle Conceptuel de Données) réalisé en PlantUML
- MLD (Modèle Logique de Données) correspondant, prêt pour PostgreSQL

## Phase 2 : Implémentation et évolution

- Script SQL d'initialisation complet
- Évolution : ajout de la traçabilité complète des anomalies (cause et impact financier)
- Compatible avec les données existantes

## Phase 3 : Collaboration et versionnement

- Projet géré via Git et GitHub
- Organisation en branches pour chaque fonctionnalité
- Utilisation des issues et GitHub Projects pour suivre les tâches
- Fusion via Pull Requests avec revue croisée
- Historique Git clair et commits liés aux issues (ex: Fixes #12)
